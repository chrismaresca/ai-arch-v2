1. Create your environment file:


   ```env

   # Social Links (Optional)
   GITHUB_URL=<your-github-url>
   LINKEDIN_URL=<your-linkedin-url>
   X_URL=<your-x-url>
   EMAIL=<your-email>
   ```
